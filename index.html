Annotation

Universal Bidirectional Activation-based learning (UBAL) [1] is a novel neural network model, which is based on the workings of biological neurons. Instead of propagating error derivatives, it uses local activation values for updating its weights. It extends other bio-inspired alternatives to error backpropagation, such as the contrastive Hebbian learning and GeneRec [2], which use presynaptic and postsynaptic activation values for the updates. The model is fully bidirectional and uses two different weight matrices for each activation direction. So far, the model has been shown to yield performance comparable to a standard fully connected MLP on the classical MNIST benchmark [1]. However, to solve the task of image classification, a convolutional architecture should be utilized. Convolutional neural networks [3] are usually better suited for processing images; therefore, the hypothesis is that Convolutional UBAL will also yield better results in the image classification task compared to the current, fully connected version. Since the UBAL model does not perform standard gradient-based learning and is fully bidirectional, the prospect of making it into a convolutional neural network brings multiple challenges, such as solving the problem of reversing the convolutional processing akin to auto-encoder networks [3]. To study the properties of the UBAL model and its convolutional version, it is necessary to continue systematic development, research, and experimental testing.
